Detailed Description

Structure containing visualisation / statistics data for a given track.

Member Data Documentation

double EfTrackInfo::angles[3]

Roll [0], Pitch [1], Yaw [2] angle of the detected face. Pitch angle not available.

EfBool EfTrackInfo::attention_now

Estimate of whether the person looks at the device in the current frame. Based oon face angle.

double EfTrackInfo::attention_time

Estimate of time [seconds] the person looked at the device. Based on face angle. (See the Developer's Guide)

double EfTrackInfo::current_time

Time of the last tracker update [seconds].

int EfTrackInfo::detection_index

Use in Expert API only. -1 if no detection in current frame. Tells the index to EfDetectionArray structure outputted by efRunFaceDetector, so that EfTrackInfo can be matched with particular EfDetection.

double EfTrackInfo::energy

Fade-out energy of the track: 1 if the face was detected in the current frame, less than 1 if not.

EfFaceAttributes EfTrackInfo::face_attributes

Face attributes (age, gender, emotion, etc.) storage.

EfBoundingBox EfTrackInfo::image_position

Face position in image pixels. Internally aggregated over past frames.

EfLandmarks EfTrackInfo::landmarks

Landmark points. Not computed by default. Point 0-7 can be further stabilized using landmark_precise_use_precise (See the Developer's Guide).

unsigned int EfTrackInfo::person_id

Person identity unique number based on face recognition, 0 if not known yet.

double EfTrackInfo::start_time

Track start time [seconds].

EfTrackStatus EfTrackInfo::status

Current status of this track.

double EfTrackInfo::total_time

Track duration [seconds].

unsigned int EfTrackInfo::track_id

Track unique number. Track is a space-time aggregation of face detecions.

double EfTrackInfo::world_position[2]

Groundplane real-world position relative to camera - [0] -> left-right, [1] -> forward-backward. In meters. Internally aggregated over past frames. Depends linearly on camera FOV.
